## 1️⃣ 實驗原理

### 🎯 什麼是細菌轉化？

**細菌轉化（Bacterial Transformation）** 是指將外源 DNA（通常是質體 plasmid）引入細菌細胞的過程。轉化後的細菌會獲得新的遺傳特性，並能複製外源 DNA。

### 🔄 轉化的三種方式

| 方式 | 說明 | 應用 |
|------|------|------|
| **自然轉化（Natural Transformation）** | 細菌自然攝取環境中的 DNA | 某些細菌物種（如 Streptococcus pneumoniae） |
| **化學轉化（Chemical Transformation）** | 使用 CaCl₂等化學物質處理細菌 | 最常用於實驗室 |
| **熱擊轉化（Heat Shock Transformation）** | 高溫短暫處理使細胞膜通透性增加 | 化學轉化的延伸 |
| **電穿孔（Electroporation）** | 使用高壓電脈衝 | 高效率轉化難轉化細菌 |

### 🔬 實驗原理圖

```
┌─────────────────────────────────────────────────────────┐
│                   細菌轉化流程                          │
├─────────────────────────────────────────────────────────┤
│  1. 準備感受態細胞（Competent Cells）                   │
│       ↓                                                  │
│  2. 加入重組質體 DNA                                    │
│       ↓                                                  │
│  3. 冰浴（Ice Bath）30 分鐘                              │
│       ↓                                                  │
│  4. 熱擊（Heat Shock）42°C 45-90 秒                      │
│       ↓                                                  │
│  5. 恢復培養（Recovery）37°C 1 小時                      │
│       ↓                                                  │
│  6. 塗佈選擇性培養基                                    │
│       ↓                                                  │
│  7. 培養並觀察菌落                                      │
└─────────────────────────────────────────────────────────┘
```

### 🧪 關鍵概念

- **感受態細胞（Competent Cells）：** 能夠攝取外源 DNA 的細菌細胞
- **質體（Plasmid）：** 環狀 DNA 分子，可獨立於染色體複製
- **抗藥性標記（Antibiotic Marker）：** 如氨苄青霉素抗性等，用於篩選轉化子
- **報告基因（Reporter Gene）：** 如 GFP（綠色螢光蛋白），用於視覺化表達

## 2️⃣ 實驗材料

### 🧫 實驗試劑

| 試劑 | 濃度/規格 | 用途 |
|------|-----------|------|
| **感受態細胞** | E. coli DH5α / TOP10 | 轉化宿主 |
| **重組質體** | pGLO / pUC19 等 | 載體 DNA |
| **CaCl₂溶液** | 100 mM, 冰凍 | 製作感受態細胞 |
| **LB 培養基** | 液體/固體 | 細菌培養 |
| **氨苄青霉素** | 100 mg/mL 儲存液 | 選擇性抗生素 |
| **阿拉伯糖（Arabinose）** | 100 mg/mL | 誘導 GFP 表達（pGLO） |
| **無菌蒸餾水** | - | 稀釋 DNA |

### 🧰 實驗器材

| 器材 | 規格 | 用途 |
|------|------|------|
| **離心機** | 冷藏型 | 收集細菌 |
| **水浴鍋** | 42°C | 熱擊處理 |
| **培養箱** | 37°C | 細菌培養 |
| **超淨工作台** | - | 無菌操作 |
| **移液槍** | 10μL, 100μL, 1000μL | 液體轉移 |
| **離心管** | 1.5 mL / 50 mL | 樣本容器 |
| **培養皿** | 90 mm | 菌落培養 |
| **塗布棒** | 玻璃/一次性 | 塗佈細菌 |
| **冰盒** | - | 保持低溫 |

### 📊 實驗耗材清單

```
□ 感受態細胞（E. coli DH5α）x 1 管
□ 質體 DNA（pGLO 或其他）x 1 管
□ LB 培養基（液體）x 10 mL
□ LB 瓊脂平板（+ Amp）x 3 個
□ LB 瓊脂平板（- Amp）x 1 個
□ 100 mM CaCl₂溶液 x 1 瓶
□ 100 mg/mL 氨苄青霉素 x 1 瓶
□ 100 mg/mL 阿拉伯糖 x 1 瓶
□ 無菌蒸餾水 x 1 瓶
```

## 3️⃣ 實驗步驟

### 📝 步驟 1：準備感受態細胞

**若使用市售感受態細胞，可跳過此步驟**

```
1. 挑取單菌落接種於 5 mL LB 培養基（含抗生素）
2. 37°C 振盪培養過夜（12-16 小時）
3. 取 1 mL 培養液轉入 100 mL 新鮮 LB
4. 37°C 振盪培養至 OD600 = 0.4-0.6（約 2-3 小時）
5. 冰浴 30 分鐘
6. 4°C 4000 rpm 離心 10 分鐘
7. 棄上清，沉澱用 50 mL 冰凍 100 mM CaCl₂重懸
8. 冰浴 30 分鐘
9. 4°C 4000 rpm 離心 10 分鐘
10. 棄上清，用 10 mL 冰凍 CaCl₂重懸
11. 分裝至 100 μL/管，-80°C 保存
```

### 📝 步驟 2：DNA 加入與冰浴

```
在超淨工作台上操作：

1. 取 2 支 1.5 mL 離心管，標記為「+DNA」和「-DNA」
2. 每管加入 100 μL 感受態細胞（冰上操作）
3. +DNA 管：加入 1-10 μL 質體 DNA（10-100 ng）
4. -DNA 管：加入等量無菌水（對照組）
5. 輕彈混勻，冰浴 30 分鐘
```

### 📝 步驟 3：熱擊處理（Heat Shock）

```
1. 將離心管放入 42°C 水浴鍋
2. 精確計時 45-90 秒（建議 50 秒）
3. 立即取出，放回冰上 2 分鐘
4. 每管加入 900 μL 無抗生素 LB 培養基
5. 37°C 振盪培養 45-60 分鐘（恢復期）
```

### 📝 步驟 4：塗佈培養基

```
1. 將培養基預熱至室溫
2. 取 100-200 μL 菌液塗佈於含抗生素的 LB 平板
3. 使用無菌塗布棒均勻塗開
4. 平板倒置（蓋子在下），37°C 培養 12-16 小時
```

### 📝 步驟 5：結果觀察

```
預期結果：

+DNA + Amp 平板：生長菌落（轉化成功）
+DNA - Amp 平板：生長菌落（細胞活性對照）
-DNA + Amp 平板：無菌落（抗藥性對照）
-DNA - Amp 平板：生長菌落（細胞活性對照）
```

## 4️⃣ 結果分析

### 📊 轉化效率計算

**轉化效率（Transformation Efficiency）=** 轉化子數量（CFU）/ DNA 用量（μg）

```
計算公式：
轉化效率 = (菌落數 × 稀釋倍數) / DNA 用量 (μg)

範例：
- 加入 10 ng DNA（0.01 μg）
- 塗佈 100 μL（總體積 1 mL，稀釋倍數 10）
- 長出 50 個菌落

轉化效率 = (50 × 10) / 0.01 = 50,000 CFU/μg
```

### 🎯 轉化效率標準

| 細胞類型 | 轉化效率範圍 | 說明 |
|----------|--------------|------|
| **優質感受態細胞** | 10⁶ - 10⁹ CFU/μg | 適合一般實驗 |
| **高效感受態細胞** | 10⁹ - 10¹⁰ CFU/μg | 適合難轉化質體 |
| **市售化學感受態細胞** | 10⁶ - 10⁸ CFU/μg | 品質穩定 |
| **電穿孔感受態細胞** | 10¹⁰ - 10¹¹ CFU/μg | 最高效率 |

### 🔬 pGLO 實驗特殊觀察

若使用 pGLO 質體（含 GFP 基因）：

| 平板條件 | 預期結果 | 說明 |
|----------|----------|------|
| **+LB/+Amp/+Arabinose** | 綠色螢光菌落 | GFP 表達 |
| **+LB/+Amp/-Arabinose** | 白色菌落 | GFP 不表達 |
| **+LB/-Amp/-Arabinose** | 白色菌落（大量） | 細胞活性對照 |

**螢光觀察條件：**
- 使用紫外線（UV）或藍光激發
- GFP 在 488 nm 激發，509 nm 發射

## 5️⃣ 常見問題與解決方案

### ❌ 問題 1：無菌落生長

| 可能原因 | 解決方案 |
|----------|----------|
| 抗生素濃度過高 | 降低至推薦濃度（Amp: 50-100 μg/mL） |
| 熱擊時間過長 | 縮短至 45-60 秒 |
| 感受態細胞活性低 | 使用新鮮或市售高品質細胞 |
| 培養時間不足 | 延長培養至 16-24 小時 |
| 培養基未加抗生素 | 確認抗生素添加正確 |

### ❌ 問題 2：菌落過多（無法計數）

| 可能原因 | 解決方案 |
|----------|----------|
| DNA 用量過多 | 減少至 1-10 ng |
| 塗佈菌液過多 | 減少至 50-100 μL |
| 轉化效率過高 | 增加稀釋倍數 |

### ❌ 問題 3：背景菌落（-DNA 組有生長）

| 可能原因 | 解決方案 |
|----------|----------|
| 感受態細胞自發抗藥性 | 更換感受態細胞批次 |
| 抗生素失效 | 使用新配製抗生素 |
| 污染 | 嚴格無菌操作 |

### ❌ 問題 4：轉化效率低

| 可能原因 | 解決方案 |
|----------|----------|
| 感受態細胞冰上放置過久 | 保持冰上操作，快速完成 |
| 熱擊溫度不準確 | 校準水浴鍋溫度 |
| 恢復培養時間不足 | 延長至 60 分鐘 |
| DNA 品質差 | 使用高純度質體 DNA（A260/A280 > 1.8） |

## 6️⃣ 實驗安全與注意事項

### ⚠️ 生物安全

1. **實驗結束後焚燒或高壓滅菌**所有受污染耗材
2. **使用生物安全等級 1（BSL-1）細菌**（如 E. coli DH5α）
3. **戴手套操作**，避免直接接觸細菌培養物
4. **實驗廢棄液**需經滅菌處理後再排放

### 🧤 無菌操作要點

1. 超淨工作台使用前以 70% 酒精擦拭
2. 火焰滅菌塗布棒
3. 培養皿開口時間盡量短
4. 避免說話、咳嗽等污染風險

### 🌡️ 溫度控制

1. **冰浴**：保持細胞活性
2. **熱擊**：精確控制時間與溫度
3. **培養**：37°C 為標準溫度
