## 1. 📖 實驗原理

### 1.1 什麼是細菌轉化？

**定義：** 細菌轉化（Bacterial Transformation）是指將外源 DNA 直接引入細菌細胞，使細菌獲得新的遺傳性状的過程。

**三大基因轉移方式：**
```
┌─────────────────────────────────────────────────────────────┐
│                    水平基因轉移 (HGT)                        │
├──────────────┬──────────────┬───────────────────────────────┤
│   轉化       │   接合       │          轉導                 │
│ (Transformation) │ (Conjugation) │      (Transduction)       │
├──────────────┼──────────────┼───────────────────────────────┤
│ 外源 DNA 直接  │ 細菌間直接   │ 噬菌體介導 DNA 轉移            │
│ 進入細菌     │ 接觸轉移     │                           │
│              │              │                           │
│ 自然或人工   │ 需性菌毛     │ 噬菌體感染                  │
└──────────────┴──────────────┴───────────────────────────────┘
```

### 1.2 轉化的應用

| 應用領域 | 說明 |
|----------|------|
| **基因克隆** | 大量複製特定 DNA 片段 |
| **蛋白質表達** | 生產重組蛋白質（如胰島素） |
| **基因功能研究** | 研究基因功能與調控 |
| **基因組編輯** | CRISPR 等基因編輯工具導入 |
| **生物技術** | 生產藥物、疫苗、工業酵素 |

### 1.3 實驗機制

**關鍵要素：**
1. **質粒（Plasmid）** - 自複製的環狀 DNA，攜帶目的基因
2. **感受態細胞（Competent Cells）** - 能吸收外源 DNA 的細菌細胞
3. **選擇標記（Selection Marker）** - 抗藥性基因（如 Ampicillin、Kanamycin）
4. **報告基因（Reporter Gene）** - GFP、LacZ 等可視化標記

**轉化方法：**
```
化學轉化（Chemical Transformation）
├─ CaCl₂ 處理法（最常見）
├─ RbCl/KCl 法
└─ 商業感受態細胞

熱激轉化（Heat Shock Transformation）
├─ 42°C 短時間熱激
└─ 形成 DNA 進入通道

電穿孔轉化（Electroporation）
├─ 高壓電脈衝
└─ 適用於難轉化菌株
```

## 2. 🔬 實驗材料與設備

### 2.1 實驗材料

| 材料 | 規格 | 用途 |
|------|------|------|
| **大腸桿菌（E. coli）** | DH5α, TOP10, JM109 | 感受態細胞 |
| **質粒 DNA** | pGLO, pUC19, pBR322 等 | 目的基因載體 |
| **LB 培養基** | LB Broth / LB Agar | 細菌培養 |
| **抗藥性抗生素** | Ampicillin (100 μg/mL) | 選擇培養 |
| **CaCl₂** | 0.1 M, ice-cold | 感受態細胞製備 |
| **甘油** | 50% | 細胞保存 |
| **阿拉伯糖** | 0.2% | pGLO 誘導劑 |

### 2.2 實驗設備

| 設備 | 規格 | 用途 |
|------|------|------|
| **離心機** | 4°C 預冷 | 收集細菌細胞 |
| **水浴鍋** | 37°C, 42°C | 培養與熱激 |
| **冷凍櫃** | -80°C | 感受態細胞保存 |
| **超淨工作台** | 無菌環境 | 無菌操作 |
| **培養箱** | 37°C | 細菌培養 |
| **移液槍** | 10 μL - 1000 μL | 液體轉移 |
| **離心管** | 1.5 mL, 50 mL | 樣本處理 |
| **培養皿** | 90 mm | 平板培養 |

### 2.3 實驗試劑配置

**1. LB 培養基（1 L）**
```
- LB Broth Powder: 25 g
- 蒸餾水：1000 mL
- 滅菌：121°C, 15 分鐘
```

**2. 抗藥性 LB 平板**
```
- LB Agar: 20 g
- 蒸餾水：1000 mL
- 滅菌後冷卻至 55°C
- 加入抗生素（如 Ampicillin 100 μL of 100 mg/mL）
- 倒平板，室溫乾燥
```

**3. 0.1 M CaCl₂溶液**
```
- CaCl₂·2H₂O: 14.7 g
- 蒸餾水：1000 mL
- 過濾除菌，4°C 保存
```

## 3. 📝 實驗步驟

### 3.1 實驗流程圖

```
┌──────────────────────────────────────────────────────────────┐
│                     細菌轉化實驗流程                         │
├──────────────────────────────────────────────────────────────┤
│                                                              │
│  1. 感受態細胞準備 → 2. DNA 加入 → 3. 冰浴 → 4. 熱激       │
│                                                              │
│       ↓                        ↓              ↓              │
│                                                              │
│  5. 冰浴 → 6. 加入培養基 → 7. 培養 → 8. 塗佈平板        │
│                                                              │
│       ↓                        ↓              ↓              │
│                                                              │
│  9. 培養觀察 → 10. 結果分析                                 │
│                                                              │
└──────────────────────────────────────────────────────────────┘
```

### 3.2 詳細步驟

#### **步驟 1：感受態細胞準備** 🧊

**方法 A：使用商業感受態細胞（推薦）**
```
1. 從 -80°C 取出感受態細胞
2. 放置在冰上緩慢解凍
3. 避免反覆凍融
4. 立即使用或分裝保存
```

**方法 B：自行製備感受態細胞**
```
1. 培養 E. coli 至 OD600 = 0.4-0.6（對數生長期）
2. 4°C 離心 5000 rpm, 10 分鐘，收集細胞
3. 棄上清，用冰冷的 0.1 M CaCl₂緩慢重懸
4. 冰浴 30 分鐘
5. 4°C 離心 5000 rpm, 10 分鐘
6. 棄上清，用 0.1 M CaCl₂（含 15% 甘油）重懸
7. 分裝，-80°C 保存
```

---

#### **步驟 2：加入 DNA** 🧬

```
1. 取 50-100 μL 感受態細胞於無菌離心管
2. 加入 1-10 μL 質粒 DNA（1-100 ng）
3. 輕輕混勻（勿振盪）
4. 設置陰性對照（不加 DNA）
```

**DNA 濃度建議：**
| 用途 | 建議 DNA 量 |
|------|------------|
| 一般轉化 | 1-10 ng |
| 低效率轉化 | 50-100 ng |
| 質粒建構驗證 | 10-50 ng |

---

#### **步驟 3：冰浴** ❄️

```
1. 將混合液置於冰上
2. 冰浴 20-30 分鐘
3. 讓 DNA 與細胞膜結合
```

---

#### **步驟 4：熱激** 🔥

```
1. 準備 42°C 水浴鍋（精確控制溫度）
2. 將離心管迅速移入 42°C 水浴
3. 熱激 45-90 秒（依菌株調整）
4. 立即移回冰上，冰浴 2 分鐘
```

**熱激時間參考：**
| 菌株類型 | 熱激時間 |
|----------|----------|
| DH5α | 45 秒 |
| TOP10 | 45 秒 |
| 效率較低菌株 | 90 秒 |

---

#### **步驟 5：恢復培養** 🍶

```
1. 加入 500 μL - 1 mL LB 培養基（無抗生素）
2. 37°C，200 rpm 震盪培養 45-60 分鐘
3. 讓細菌恢復並表達抗藥性基因
```

---

#### **步驟 6：塗佈平板** 🎨

```
1. 離心細胞（可選）：5000 rpm, 1 分鐘，棄部分上清
2. 將細胞懸液塗佈於抗藥性 LB 平板
3. 使用無菌塗棒均勻塗開
4. 倒置培養皿，37°C 培養 12-16 小時
```

**塗佈技巧：**
- 使用 L 型塗棒（酒精燈火焰滅菌）
- 塗佈時轉動培養皿
- 避免劃破瓊脂表面
- 多餘液體可倒出或吸乾

---

#### **步驟 7：培養觀察** 👁️

```
1. 37°C 培養 12-16 小時
2. 觀察菌落生長情況
3. 記錄菌落數量與形態
4. 如有報告基因，觀察顏色/螢光變化
```

---

### 3.3 實驗對照組設置

| 對照組 | 內容 | 預期結果 | 目的 |
|--------|------|----------|------|
| **陽性對照** | 感受態細胞 + 已知質粒 | 大量菌落 | 驗證感受態細胞活性 |
| **陰性對照 1** | 感受態細胞 + 無 DNA | 無菌落 | 驗證抗生素有效性 |
| **陰性對照 2** | 未處理細胞 + 質粒 | 無菌落 | 驗證轉化效率 |
| **實驗組** | 感受態細胞 + 目的質粒 | 菌落生長 | 測試轉化效率 |

## 4. 📊 結果分析

### 4.1 轉化效率計算

**公式：**
```
轉化效率 = (菌落數 × 總體積) / (DNA 量 × 塗佈體積)

單位：transformants/μg DNA
```

**範例計算：**
```
- 加入 DNA 量：10 ng (0.01 μg)
- 總培養液體積：1000 μL
- 塗佈體積：100 μL
- 計數菌落數：150 個

轉化效率 = (150 × 1000) / (0.01 × 100)
         = 150,000 transformants/μg DNA
```

### 4.2 轉化效率評價

| 效率等級 | 轉化效率 (transformants/μg) | 說明 |
|----------|----------------------------|------|
| **優秀** | > 10⁷ | 商業感受態細胞水平 |
| **良好** | 10⁶ - 10⁷ | 自行製備可達水準 |
| **可接受** | 10⁵ - 10⁶ | 基本實驗可用 |
| **需改進** | < 10⁵ | 檢查實驗條件 |

### 4.3 結果判讀

```
┌────────────────────────────────────────────────────────────┐
│                    結果判讀指南                            │
├────────────────────────────────────────────────────────────┤
│                                                            │
│  陰性對照有菌落 → 抗生素失效或污染                         │
│                                                            │
│  陽性對照無菌落 → 感受態細胞失去活性                       │
│                                                            │
│  實驗組無菌落 → DNA 問題或轉化失敗                         │
│                                                            │
│  菌落數太少 → 轉化效率低，檢查各步驟                       │
│                                                            │
│  菌落數太多 → 適當稀釋或減少塗佈量                         │
│                                                            │
└────────────────────────────────────────────────────────────┘
```

## 5. ⚠️ 注意事項

### 5.1 無菌操作

**超淨台使用規範：**
- ✅ 使用前紫外線滅菌 30 分鐘
- ✅ 酒精擦拭工作台面
- ✅ 所有操作在火焰附近進行
- ✅ 避免手部跨越無菌區域
- ✅ 定期滅菌塗棒與移液槍頭

### 5.2 溫度控制

| 步驟 | 溫度 | 重要性 |
|------|------|--------|
| CaCl₂處理 | 0-4°C | ⭐⭐⭐⭐⭐ 必需 |
| 冰浴 | 0-4°C | ⭐⭐⭐⭐⭐ 必需 |
| 熱激 | 42°C ± 0.5°C | ⭐⭐⭐⭐⭐ 關鍵 |
| 恢復培養 | 37°C | ⭐⭐⭐⭐ 重要 |
| 平板培養 | 37°C | ⭐⭐⭐ 一般 |

### 5.3 常見問題與解決方案

| 問題 | 可能原因 | 解決方案 |
|------|----------|----------|
| **無菌落生長** | 感受態細胞失活 | 使用新製備細胞 |
| | DNA 降解 | 檢查 DNA 品質 |
| | 抗生素失效 | 更換新抗生素 |
| | 熱激時間不足 | 調整熱激時間 |
| **菌落數太少** | 轉化效率低 | 使用高效率感受態細胞 |
| | DNA 濃度低 | 增加 DNA 量 |
| | 培養時間不足 | 延長培養時間 |
| **菌落形態異常** | 污染 | 嚴格無菌操作 |
| | 抗生素濃度不當 | 優化抗生素濃度 |

### 5.4 安全注意事項

**生物安全：**
- ⚠️ 使用合適的生物安全等級（通常 BSL-1）
- ⚠️ 廢棄物滅菌後處理
- ⚠️ 實驗後徹底洗手
- ⚠️ 避免產生氣溶膠

**化學安全：**
- ⚠️ 抗生素可能致敏，戴手套操作
- ⚠️ 酒精燈使用注意安全
- ⚠️ 高溫設備防燙傷
