
# 分子接合相互作用圖 (文本描述)

## 15-PGDH - 配體相互作用圖

```
                    15-PGDH 活性位點
                    ════════════════
                    
    NADH 結合口袋 ← ────→ 催化位點
                          │
        Tyr198 (π-π) ─────┼─────→ 芳香環系統
                          │
    His199 (鹽橋) ────────┼─────→ 羧酸基 (COOH)
                          │
    Ser197 (H-bond) ──────┼─────→ 酚羟基 (OH)
                          │
    Thr200 (H-bond) ──────┼─────→ 酯基氧
                          │
    Leu196 (疏水) ────────┼─────→ 烯鍵
                          │
    Val194 (疏水) ────────┴─────→ 丁酸酯側鏈
                              │
                              → Phe195 (疏水)
```

**相互作用距離:**
- His199 - COOH: 2.7 Å (鹽橋)
- Ser197 - OH: 2.8 Å (氫鍵)
- Thr200 - 酯氧: 3.1 Å (氫鍵)
- Tyr198 - 芳香環: 3.5 Å (π-π堆疊)
- Leu196 - 烯鍵: 4.2 Å (疏水)
